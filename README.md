uib-visste-du
=============

Shows fun-facts about a random study-programme at the University of Bergen (UiB).
NB! Not all study programmes have entered a fun-fact in UiB's systems.

Dependencies
------------
  
jQuery for Javascript.

Uses uibStudyInfo for back-end.
